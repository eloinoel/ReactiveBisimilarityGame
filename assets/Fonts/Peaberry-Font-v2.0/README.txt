ABOUT PEABERRY

A clean and bold pixel font family inspired by geometric fonts like GT Walsheim.
Includes regular, double spaced, and monospaced variants.
Pixel perfect at multiples of 16px.

WHAT'S INCLUDED

1. @font-face pack with demo .html and .css stylesheet.
Font file formats: .ttf, .otf, .eot, .svg, .woff, .woff2

2. Bitmap font pack with 6 freebie bitmap fonts made using Peaberry.
Bitmap file formats: .png, .txt, .xml, .lua, .skf, .fnt  BMFont Text, .fnt - XML)
- See the ABOUT pdf in the bitmap font folder for more info, e.g. which file(s) to use with different platforms.

** BONUS **
GoldPeaberry includes a normal map that can be used to add realtime lighting effects to bitmap text. The normal map and bitmap texture file are pre-packed in the included atlas (GoldPeaberry.json).
- See the HELP pdf in the GoldPeaberry folder for more info.
